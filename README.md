# Fee Payment and Student Record System

A desktop-based application developed using **Java Swing, JDBC, and MySQL** to manage student records and fee payments efficiently.

## Features

* Secure Login Authentication
* Add / Update / Delete Student Records
* Fee Payment Tracking
* MySQL Database Integration
* Simple and Interactive GUI

## Technologies Used

* Java Swing
* JDBC
* MySQL
* IntelliJ IDEA

## Run the Project

Download the JAR file and run:

java -jar "jdbc D.jar"
